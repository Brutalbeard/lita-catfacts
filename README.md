**lita-catfacts** is a handler for [Lita](https://github.com/jimmycuadra/lita) that provides very interesting facts about cats.

## Installation

Add lita-catfacts to your Lita instance's Gemfile:

``` ruby
gem "lita-catfacts"
```

## Usage

    Lita: catfacts
    Lita: catfact
